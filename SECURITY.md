# 安全性政策

我們非常重視「實人教育 少年導覽隊」專案的安全性與使用者隱私。由於本網頁涉及兒少公益行動與學校教育計畫，確保程式碼與部署環境不洩漏任何敏感資料是我們的首要任務。

## 已完成的安全優化

在本次重構版本中，我們已針對過往已知的安全風險進行了全面排查與修正：
1. **隱私與憑證清理：** 確保原始程式碼、註解、設定檔中無任何後台 API Key、金鑰、伺服器路徑或個人敏感資訊。
2. **消滅 XSS 漏洞風險：** 嚴格限制或移除 AI 生成程式碼中濫用 `innerHTML` 直接渲染未過濾字串的作法，一律改用安全的 `textContent` 或 DOM 節點建立方法。
3. **第三方組件檢視：** 本網頁所內嵌之 Google Search Console 驗證標籤（Meta Tag）與 SurveyCake 捐款問卷連結，均已確認來源合法性，無供應鏈劫持風險。

## 報告資安漏洞

如果你在重構後的程式碼、圖片資源或部署的 GitHub Pages 中發現任何安全性漏洞（例如敏感資訊殘留、跨站腳本攻擊風險等）：

1. **請勿** 建立公開的 GitHub Issue 來回報漏洞。
2. 請直接發送私人電子郵件至：`pig0911@shininglife-edu.com` 進行不公開通報。
3. 我們將會在收到通報後的 **48 小時內** 進行技術評估，確認後將直接於網頁端進行修復修補（Patch），並在 Commit 中向通報者致謝。

感謝共同守護兒少教育專案的網路安全！
